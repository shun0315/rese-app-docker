## 開発環境構築について

Docker を用いて、LEMP 開発環境を構築できます。※ローカル環境のみ

## インストール方法

#### 任意のディレクトリに移動

```
$ cd [任意のディレクトリ]
```

#### Github から Clone

```
$ git clone https://github.com/shun0315/rese-app-docker.git
```

#### src ディレクトリを作成

```
$ mkdir src
```

#### src ディレクトリに移動

```
$ cd src
```
